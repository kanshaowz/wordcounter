# WordCounter - 字数统计

一个极简风格的字数统计工具，专注于写作体验。

## 🌟 特色

- **灰色极简设计** - 深色主题，减少视觉干扰，专注写作
- **实时统计** - 字数、字符数、段落数、阅读时间
- **写作目标** - 设定并追踪写作进度
- **多语言支持** - 中文、英语、西班牙语、日语、韩语
- **专注模式** - Ctrl/Cmd + Shift + F 切换专注模式

## 🚀 在线使用

访问: https://kanshaowz.github.io/wordcounter

## 🛠 功能

- **字数统计** - 智能识别不同语言的字数计算方式
  - 中日韩：按字符数计算
  - 英语/西班牙语：按单词数计算
  
- **写作目标** - 设置目标字数，实时显示进度条

- **阅读时间** - 根据内容长度估算阅读时间

## 📝 快捷键

| 快捷键 | 功能 |
|--------|------|
| Ctrl/Cmd + Shift + F | 切换专注模式 |
