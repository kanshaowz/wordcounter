# WordCounter Pro - 专业写作工具

一个专为写作者设计的极简风格字数统计工具，集成了写作目标管理、统计图表、多文档管理等专业功能。

![Version](https://img.shields.io/badge/version-2.0.0-blue)
![License](https://img.shields.io/badge/license-MIT-green)

## ✨ 特色功能

### 1. 写作目标进度可视化
- 设置字数/字符目标
- 实时进度条显示
- 环形图表可视化
- 剩余字数和预计完成时间

### 2. 语法高亮
- Markdown 语法实时高亮
- 标题、粗体、斜体、代码、链接等样式识别
- 写作时即时预览效果

### 3. 写作统计图表
- 字数趋势折线图
- 实时写作速度统计
- 总字数、总字符、写作时长、平均速度
- 历史数据记录

### 4. 导出功能
- 导出为 TXT 格式
- 导出为 Markdown 格式
- 一键下载当前文档

### 5. 专注模式
- 全屏沉浸式写作
- 隐藏所有干扰元素
- F11 快捷键切换
- 鼠标悬停显示工具栏

### 6. 自动保存
- 实时自动保存（每5秒）
- 本地存储，无需注册
- 保存状态可视化指示器
- 关闭页面时提醒未保存内容

### 7. 多文档管理
- 侧边栏文档列表
- 创建、切换、删除文档
- 文档字数和修改时间显示
- 快捷键新建文档 (Ctrl/Cmd + N)

### 8. 灰色极简风格
- 精心调校的灰色系配色
- 无干扰的编辑界面
- 流畅的过渡动画
- 响应式设计，支持移动端

## 📝 功能详情

### 字数统计
智能识别不同语言的字数计算方式：
- 中文、日文、韩文：按字符数计算
- 英文、西班牙文：按单词数计算

### 阅读时间估算
根据内容长度智能估算阅读时间：
- 中文：约 400 字/分钟
- 英文：约 200 词/分钟

### 快捷键

| 快捷键 | 功能 |
|--------|------|
| F11 | 切换专注模式 |
| Ctrl/Cmd + S | 手动保存 |
| Ctrl/Cmd + N | 新建文档 |
| ESC | 关闭弹窗/退出专注模式 |

## 📦 本地运行

```bash
# 克隆仓库
git clone https://github.com/kanshaowz/wordcounter.git

# 进入目录
cd wordcounter

# 启动本地服务器 (任选其一)
# Python 3
python -m http.server 8000

# Node.js
npx serve

# 访问 http://localhost:8000
```
